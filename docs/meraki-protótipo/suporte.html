Como é libertado o pagamento do Escrow?

Quando for submetido um Milestone (Entrega), a empresa aprova e os fundos correspondentes para esse Milestone são libertados de imediato para a Wallet do especialista via nossa API de pagamentos integrada com RabbitMQ.

Taxas cobradas pela plataforma

O Meraki cobra uma comissão fixa de operação técnica (10% sobre o valor gerado aos dev) e para empresa existe zero comissão extra base. As transações seguras são asseguradas criptograficamente.

E se houver litígio ou disputa técnica?

Pode solicitar uma auditoria do código por um Tech Lead neutral nomeado pela casa, enviando o caso diretamente para o nosso Helpdesk usando o botão ao lado.
